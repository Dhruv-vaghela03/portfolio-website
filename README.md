# Responsive Portfolio Website

Personal portfolio website built with **HTML5, CSS3, and JavaScript**.

## Features
✅ Fully Responsive Design (Mobile + Tablet + Desktop)
✅ Modern UI with Gradient Header
✅ Smooth Hover Animations
✅ Project Showcase Section
✅ Skills Display
✅ Contact Information

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Design:** Responsive Web Design, Flexbox, CSS Grid
- **Hosting:** GitHub Pages

## Live Demo
[View Live](https://dhruv-vaghela03.github.io/portfolio-website/)

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. That's it! No dependencies needed.
